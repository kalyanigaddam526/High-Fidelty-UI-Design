# High-Fidelty-UI-Design 
# Project Title
High-Fidelity UI Design for E-Commerce Mobile App
Project Overview
This project transforms a low-fidelity wireframe into a visually appealing high-fidelity mobile application design. The design focuses on improving user experience through modern UI elements, attractive color schemes, intuitive navigation, and consistent visual hierarchy.
# Tools Used
Figma
Google Fonts (Poppins)
Material Design Icons

# Color Palette
